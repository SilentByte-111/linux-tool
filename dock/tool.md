# 全球命令管理工具 (tool)
[返回目录](dock.md)
## 工具介绍

全球命令管理工具 (tool) 是一个方便用户将自定义 shell 脚本安装为全局命令的实用工具。它提供了一个菜单界面，让用户可以轻松地管理自己的命令脚本，支持安装、修改、卸载和列出所有已安装的命令。

## 功能特性

1. **安装新的 .sh 文件为全局命令**
   - 支持相对路径和绝对路径
   - 自动从文件名生成默认命令名
   - 可自定义命令说明

2. **修改已有命令名或说明**
   - 可以更改已安装命令的名称
   - 可以更新命令说明

3. **卸载全局命令**
   - 移除已安装的全局命令

4. **列出所有命令及说明**
   - 显示所有已安装的命令及其说明

## 使用方法

### 安装命令
1. 运行 \`tool\` 启动工具
2. 选择选项 \`1\` - 选择并安装 .sh 文件为全局命令（支持相对/绝对路径）
3. 输入要安装的 .sh 文件路径（支持相对路径和绝对路径）
4. 输入希望使用的命令名（回车使用默认文件名）
5. 输入命令说明（支持中文）

### 修改命令
1. 选择选项 \`2\` - 修改已有命令名或说明
2. 输入要修改的命令名
3. 输入新的命令名（回车保持不变）
4. 输入新的说明（回车保持不变）

### 卸载命令
1. 选择选项 \`3\` - 卸载全局命令
2. 输入要卸载的命令名

### 列出命令
1. 选择选项 \`4\` - 列出所有命令及说明

## 命令安装位置

所有安装的命令都放在 \`/usr/local/bin\` 目录下。

## 说明文件

命令说明存储在 \`/usr/local/bin/.tool_descriptions\` 文件中。

## 命令命名规则

命令名只能包含字母、数字、下划线和连字符。

## 注意事项

1. 由于需要复制文件到系统目录，需要 root 权限（sudo）
2. 建议为命令提供有意义的说明，便于后续管理
3. 定期使用选项 \`4\` 检查已安装的命令
4. 请确保安装的脚本是安全的

## 常见问题

**Q: 为什么我的脚本无法安装？**
A: 请确保脚本文件存在且具有 .sh 扩展名。

**Q: 如何使用相对路径？**
A: 输入类似 \`./script.sh\` 或 \`../folder/script.sh\` 的路径。

**Q: 如何使用绝对路径？**
A: 输入类似 \`/home/user/script.sh\` 的完整路径。

**Q: 命令安装后如何使用？**
A: 安装完成后，可以在任何目录下直接使用命令名运行脚本。
